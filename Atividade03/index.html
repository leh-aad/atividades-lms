Linguagem de marcação e script

HTML

O HTML(Hyper Text Markup Language) é uma linguagem de marcação que descreve documentos Web.
Uma linguagem de marcação é um conjunto de tags de marcação.

Elementos HTML

Composto por tudo que está entre a tag de início (inclusive) e tag de fim (inclusive).

HTML - Estrutura

Seção Head: Título, Recursos externos, Metadados
Seção Body: Texto, Imagem, Links

CSS

CSS, que significa Cascading Style Sheets, descreve como os elementos são renderizados na tela.
